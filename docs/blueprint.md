# **App Name**: Page Forge

## Core Features:

- Visual Editor: Drag-and-drop interface for adding and arranging page elements.
- Template Library: Pre-designed templates for various page types (landing, contact, about, etc.).
- Live Preview: Real-time preview of the page as it's being built.
- Code Export: Generate clean HTML, CSS, and JavaScript code for the created page.
- AI Assistant: AI-powered tool for suggesting design improvements and accessibility enhancements.
- Project Management: Project saving and loading functionality.

## Style Guidelines:

- Primary color: Teal (#4db6ac) for a clean and professional feel.
- Background color: Light gray (#f0f4c4) for a neutral backdrop.
- Accent color: Purple (#8e24aa) for highlighting key UI elements.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable look.
- Use a consistent set of minimalist icons throughout the interface.
- Maintain a clear and organized layout with consistent spacing and alignment.
- Use subtle transitions and animations to enhance the user experience.
